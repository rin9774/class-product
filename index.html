Find My Water - Product Proposal

Team Members

Ryan Tran

Application Overview

The "Find My Water" app helps users locate nearby safe and clean water sources in real-time, making it easier to access water in emergency situations or in areas with scarce resources.

Technology Overview

Languages: JavaScript, HTML, CSS

Libraries: React, Leaflet (for map integration), Axios (for API requests)

Platforms: Web (Browser-based)

Hosting: GitHub Pages

Proposal Last Updated

March 23, 2025

Feature List

  • Locate nearby safe water sources using maps
  • User-friendly interface to search and filter water sources
  • Real-time updates on water quality and availability
  • Ability to report water sources

Future Feature List

  • Push notifications for nearby water availability
  • Integration with weather data to predict water sources availability
  • Offline mode for remote areas
  • User rating system for water sources

Basic Technical Features

  • Interactive map displaying water source locations
  • Search functionality for nearby water sources
  • API integration for water source data

Advanced Technical Features

  • Real-time water quality data updates from API
  • GPS integration for user location tracking
  • Water source reporting and community updates
